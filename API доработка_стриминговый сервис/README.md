**Кейс: Поставить ТЗ для разработчика для доработки API**



**Легенда:** Заказчик (стриминговый сервис) хочет акцентировать внимание на своей особенности - представлять некоторые фильмы и сериалы эксклюзивно. И хочет дать пользователям возможность увидеть весь список фильмов и сериалов, которые представлены эксклюзивно.



**Задачи:**

1. Проанализировать текущие диаграммы UML Component Diagram, UML Sequence Diagram, API, проанализировать тело запроса и ответа, определить, какие изменения нужно внести для реализации новой функциональности.
2. Создать тикет на доработку API, описать новые примеры xml-запроса и xml-ответа.
