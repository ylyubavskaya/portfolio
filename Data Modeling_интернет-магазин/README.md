**Кейс: Создание ER-диаграммы для интернет-магазина**



Нужно составить ER-модель данных логического уровня интернет-магазина.


**Задачи:**



1. Создать пример ER-диаграммы интернет-магазина в [нотации Чена](https://github.com/ylyubavskaya/portfolio/blob/main/Data%20Modeling_%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B5%D1%82-%D0%BC%D0%B0%D0%B3%D0%B0%D0%B7%D0%B8%D0%BD/datamodeling_%D0%BD%D0%BE%D1%82%D0%B0%D1%86%D0%B8%D1%8F%20%D0%A7%D0%B5%D0%BD%D0%B0.jpg) и в [нотации Crow's Foot](https://github.com/ylyubavskaya/portfolio/blob/main/Data%20Modeling_%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B5%D1%82-%D0%BC%D0%B0%D0%B3%D0%B0%D0%B7%D0%B8%D0%BD/datamodeling_%D0%BD%D0%BE%D1%82%D0%B0%D1%86%D0%B8%D1%8F%20Crow's%20Foot.jpg) на основе интервью с заказчиком.

2. На основе спецификации создать [расширенную ER-диаграмму](https://github.com/ylyubavskaya/portfolio/blob/main/Data%20Modeling_%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B5%D1%82-%D0%BC%D0%B0%D0%B3%D0%B0%D0%B7%D0%B8%D0%BD/datamodeling_project_online_shop_3NF.png) интернет-магазина, привести ее к 3НФ.
