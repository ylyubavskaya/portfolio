**Кейс: Создание BPMN и UML State Machine диаграмм для приложения частной клиники**



**Легенда:** Сеть медицинских клиник хочет разработать приложение для пациентов. В приложении планируется реализовать функционал, который позволит пациентам регистрироваться в приложении клиники, самостоятельно находить необходимого врача, подходящее время для приёма и записываться на осмотр, минуя ручную работу администратора клиники.



**Задачи:**

1. На основе интервью с пациентами, врачами и администратором клиники создать диаграмму бизнес-процесса AS IS и диаграмму состояний.
2. Создать диаграмму бизнес-процесса TO BE с учётом автоматизации процесса записи на приём к врачу в приложении медицинской клиники.
