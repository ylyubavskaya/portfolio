**Задачи:**

- На основе интервью с пациентами, врачами и администратором клиники создать диаграмму бизнес-процесса AS IS и диаграмму состояний.
- Создать диаграмму бизнес-процесса TO BE с учётом автоматизации процесса записи на приём к врачу в приложении медицинской клиники.
