**Кейс: Разработка спецификации автоматизации подачи заявки на кредит и заключения сделки в банке. Полный цикл.**

Для подготовки спецификации велась работа над:
- Сбором и формализацией бизнес-требований и бизнес-правил: описание BPMN AS IS/ TO BE, ограничения, бизнес-цели и пр.;
- Моделированием функциональной модели системы, описанием Use Case и ВИ;
- Созданием требований для UI;
- Моделированием статусной модели сущностей "Заявка" и "Сделка";
- Разработкой алгоритима скориннга заявки;
- Моделированием данных и описанием сущностей и связей (ERD);
- Подготовкой примеров SELECT sql-запросов для поиска информации по ERD;
- Описанием общей архитектуры системы и интеграциями между сервисами: UML Sequence, описание методов, маппинг, описание обработки ошибок;
- Разработкой функциональных и нефункциональных требований.
