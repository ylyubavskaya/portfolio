**Кейс: Разработка документации API для новой функции стримингового сервиса**



**Легенда:** Заказчик (стриминговый сервис) хочет добавить новую функцию - сортировку фильмов и сериалов по рейтингу от кинокритиков.



**Задачи:**

1. Проанализировать интервью с заказчиком.
2. Проанализировать текущую диаграмму UML Sequence Diagram, изменить ее с учетом добавления новой функции.
3. Проанализировать текущую документацию API. Понять, какие изменения нужно будет внести.
4. Создать тикет на разработку API.
