**Кейс: Создание варфреймов для приложения частной клиники**



Нужно создать раскадровку для сети медицинских клиник. В распоряжении есть диаграмма потоков данных и диаграмма структуры интерфейса.



В мобильном приложении пациенты могут просмотреть список всех доступных врачей, записаться к заинтересовавшему их специалисту и проверить свои запланированные визиты. 



**Задачи:**

1. На основе DFD-диаграммы и ISD-диаграммы создать варфреймы с элементами интерфейса частной клиники.
2. Настроить интерактивный прототип.
